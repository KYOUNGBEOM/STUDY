# 제어자란?
  - 클래스, 변수, 메서드의 선언부에 함께 사용되어 부가적인 의미를 부여

# 제어자의 종류
   - 제어자는 크게 접근 제어자와 그 외의 제어자로 나뉨
   - 접근 제어자 : public, protected, dafault, private
   - 그 외 : static, final, abstract 등

# 접근 제어자
   1. public
      - 접근제한이 없음
   3. protected
      - 같은 패키지 內 + 다른  패키지의 자손 클래스
   5. default
      - 같은 패키지 內
   7. private
      - 같은 클래스 內
     
# 접근 제어자를 사용하는 이유
   - 외부로부터 데이터를 보호
   - 내부적으로만 사용되는 데이터를 감춤
